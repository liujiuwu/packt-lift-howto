CSS Selectors

This page shows a few examples of CSS selectors. Not everything is shown here, we try to concentrate on the most widely used cases to limit the confusion. 

<span data-lift="Selectors.userName"><span id="user-name">User Name Here</span></span>
becomes: User Name Here 

        <div class="expense">Item 1</div>
        <div class="expense">Item 2</div>
        <div class="expense">Item 3</div>
Let's user Selectors.addExpenseClass to add a class to all elements with class "expense": 

        <div class="expense my-extra-class">Item 1</div>
        <div class="expense my-extra-class">Item 2</div>
        <div class="expense my-extra-class">Item 3</div>

Now let's add a link to an anchor tag: 


        <a id="user-link" href="#"><span class="user-name">User Name Here</span></a>
